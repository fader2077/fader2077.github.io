醫學影像分析與多任務學習

專案概述

基於 Transformer 與多任務學習機制的醫學影像分析研究，融合輕量型 MobileViT 模組與 U-Net 架構，實現高效能的醫學影像分割與分類任務。

研究目標

  • 開發輕量化的醫學影像分析模型
  • 實現多任務學習框架，同時處理分割與分類
  • 優化模型在邊緣設備上的推理效能
  • 提升醫學診斷的準確性與效率

技術實現

U-Net Architecture MobileViT Backbone PyTorch Framework Multi-Task Learning Attention Mechanism Data Augmentation

研究成果

  • 發表於國際期刊與研討會
  • 模型準確率提升 15% 以上
  • 推理速度優化 3 倍

AI Agriculture

LLM × 智慧農業專家系統

專案概述

基於知識圖譜與 PEFT 的農業專家系統，結合 RAG (Retrieval-Augmented Generation) 與 Fine-tuning 技術，打造智慧化的農業諮詢平台。

研究目標

  • 建構農業領域知識圖譜
  • 實現高效的 RAG 檢索增強生成
  • 應用 PEFT 技術進行模型微調
  • 整合 YOLOv5 進行病蟲害識別

技術實現

Knowledge Graph RAG Pipeline PEFT / LoRA YOLOv5 LangChain Vector Database

研究成果

  • 發表於 ITAOI 2024 研討會
  • 建立超過 10,000 筆農業知識條目
  • 問答準確率達 92%
