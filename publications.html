Exploring the Integration of MobileViTv3 in U-Net for Enhanced Multi-Task Learning

Authors: Jie-Xin Li, et al.
Venue: Med++ MNIST Dataset
Year: 2024

Abstract

This paper explores the integration of MobileViTv3 as the backbone network in U-Net architecture for enhanced multi-task learning in medical image analysis. The proposed method leverages the efficiency of MobileViT while maintaining the segmentation capabilities of U-Net, achieving state-of-the-art performance on the Med++ MNIST dataset.

Lightweight GPS-Aided Recycling Analysis System in Edge Computing

Authors: Jie-Xin Li, et al.
Venue: Green Computing Conference
Year: 2024

Abstract

This paper presents a lightweight GPS-aided recycling analysis system designed for edge computing environments. The system combines YOLOv5 object detection with GPS location data to enable intelligent waste classification and recycling route optimization, promoting sustainable waste management practices.

Authors: Jie-Xin Li, et al.
Venue: Machine Learning Conference
Year: 2024

Abstract

This paper investigates the application of Data Cartography techniques to tree-based machine learning models such as XGBoost and Random Forest. The proposed method provides insights into training dynamics and data quality assessment for ensemble models, enabling better understanding of model behavior and data characteristics.
